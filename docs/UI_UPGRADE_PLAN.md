# InputMonitor UI 美化升级方案

## 目标

在不改变原有实现逻辑的前提下，重绘图标、重做页面视觉层次，并加入黑白两套主题切换。  
整体方向：亚克力质感、平面设计、多彩朦胧、统一图标语言、现代桌面工具风格。

## 视觉关键词

- 亚克力毛玻璃：半透明卡片、柔和边框、背景渐变色块
- 平面化图标：统一线宽、圆角端点、双色渐变强调
- 黑/白双主题：深色科技感 + 浅色清爽工作台
- 数据仪表盘化：指标卡片、活动分布、时间线都使用统一卡片系统
- 降低“程序味”：减少纯黑、纯灰、硬边框，增加呼吸感

## 页面结构建议

### 1. 主仪表盘

保留原逻辑：
- 今日点击数
- 今日按键数
- 活跃时间
- 当前推断活动
- 时间线
- 活动分布

视觉调整：
- 顶部增加状态胶囊：采集中 / 已暂停 / 隐私模式
- 指标卡片使用四种轻渐变图标
- 活动分布使用圆角进度条，而不是密集文本
- 时间线用胶囊块表示活动段，颜色由活动类型决定

### 2. 快速标注栏

保留原逻辑：
- 7 个预定义按钮
- 自定义活动按钮
- + 按钮内联添加

视觉调整：
- 按钮改成小型彩色胶囊
- 当前选中活动增加发光边框
- 自定义活动统一使用 `mixed` 风格图标，避免视觉混乱

### 3. 时间线页

保留原逻辑：
- 按天查看
- 日期导航
- 标注优先展示

视觉调整：
- 左右日期切换做成圆角按钮
- 每个时间段做成分层卡片
- 人工标注段增加小铅笔图标
- 空闲段降低对比度

### 4. 设置页

保留原逻辑：
- 采集控制
- 隐私
- 数据管理
- 自定义活动
- 刷新间隔
- 开机自启
- 定时提醒

视觉调整：
- 分成“采集”“隐私”“数据”“提醒”“外观”五组
- 每组用 AcrylicCard 包起来
- 开关使用统一强调色
- 新增“外观模式：深色 / 浅色 / 跟随系统（可选）”

## 活动色彩建议

| 活动 | 主色 | 视觉含义 |
|---|---|---|
| coding | #7C5CFF | 逻辑、结构 |
| writing | #FFB347 | 文本、创作 |
| chat | #29D3A6 | 沟通、即时反馈 |
| browsing | #4DB6FF | 信息流 |
| reading | #9AA7FF | 安静、低频输入 |
| gaming | #FF5C8A | 高强度操作 |
| idle | #8A93A6 | 空闲、低活跃 |
| mixed | #C77DFF | 混合状态 |

## 不建议改动的地方

- 不改采集线程
- 不改 SQLite 表结构
- 不改 5 分钟窗口聚合
- 不改规则分类优先级
- 不改托盘和单实例机制
- 不改隐私默认策略

## 建议新增的配置项

```python
appearance_mode = "dark"  # dark / light
accent_style = "aurora"   # aurora / cyan / violet
acrylic_enabled = True
compact_mode = False
```

## 集成优先级

1. 先换全局颜色和字体 token
2. 再换图标
3. 再改卡片和按钮样式
4. 最后加主题切换
